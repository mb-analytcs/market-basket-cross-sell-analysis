# Entendimento do Negócio

## Contexto

Uma rede de supermercados deseja aumentar o faturamento por meio de estratégias de cross-sell, oferecendo recomendações de produtos que costumam ser comprados juntos.

Para isso, será utilizada a técnica de Market Basket Analysis aplicada aos dados públicos do Instacart.

---

## Problema de Negócio

Atualmente, as recomendações de produtos são limitadas e não exploram os padrões reais de compra dos clientes.

O objetivo é identificar combinações frequentes de produtos para apoiar ações comerciais que aumentem o ticket médio.

---

## Objetivos da Análise

- Identificar produtos frequentemente comprados em conjunto.
- Encontrar oportunidades de cross-sell.
- Descobrir regras de associação entre produtos.
- Apoiar campanhas promocionais.
- Gerar insights para melhorar a experiência do cliente.

---

## Perguntas de Negócio

- Quais produtos são frequentemente comprados juntos?
- Quais categorias apresentam maior potencial de cross-sell?
- Quais combinações possuem maior confiança e lift?
- Existem produtos "gatilho" que estimulam compras adicionais?

---

## KPIs

- Suporte (Support)
- Confiança (Confidence)
- Lift
- Quantidade de regras encontradas
- Produtos com maior potencial de cross-sell

---

## Entregáveis

- Limpeza dos dados
- Modelagem dos dados
